GeoVault

GeoVault is a secure document access system designed to protect sensitive information using multi-layered security controls. It enables users to share files with strict conditions such as location verification, time-based access, and limited usage to prevent unauthorized access.  

By combining modern web technologies with backend services like Supabase, GeoVault ensures reliable, scalable, and real-time secure file handling. This makes it ideal for scenarios where data privacy and controlled access are critical, such as corporate, academic, or legal environments.


---

## 🚀 Features

- 📍 Location-based access (Latitude & Longitude validation)
- ⏰ Time-restricted document access
- 🔢 Limited number of file views
- 👤 Secure user authentication
- 🔐 Encrypted document handling
- ☁️ Backend powered by Supabase

---
